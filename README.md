# IT Support / Help Desk Home Lab Projects

## Overview
These Home Labs show case hands-on It support and help desk  
The labs focus on troubleshooting, user support, system administration, and documentation commonly required in entry-level IT roles.

---

## Labs Included

- Windows Active Directory Home Lab
- IT Ticketing System (ServiceNow / Jira)
- Desktop & OS Troubleshooting
- Networking Fundamentals
- Hardware & Peripheral Support

---

## Lab Details

### 1. Windows Active Directory Home Lab
**Tools:** Windows Server, Windows 10/11, VirtualBox / VMware

**What I did:**
- Installed and configured a Windows Server domain environment
- Created and managed user accounts and security groups
- Performed password resets, account unlocks, and permissions management
- Joined Windows 10/11 machines to the domain
- Applied basic Group Policy settings

**Skills Demonstrated:**
- Active Directory administration
- User account management
- Troubleshooting authentication issues
- Domain-joined workstation support

---

### 2. Desktop & OS Troubleshooting Lab
**Tools:** Windows 10/11

**What I did:**
- Installed and configured operating systems
- Resolved software crashes, driver issues, and performance problems
- Created and managed local user profiles

**Skills Demonstrated:**
- OS troubleshooting
- Software support
- Endpoint management

---

### 3. Networking Fundamentals Lab
**Tools:** Home router, Command Line

**What I did:**
- Configured DHCP and static IP addressing
- Diagnosed connectivity issues using ping, tracert, and ipconfig
- Troubleshot Wi-Fi and VPN connectivity problems

**Skills Demonstrated:**
- Networking fundamentals
- Connectivity troubleshooting
- Command-line tools
